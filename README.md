# Portfolio

A clean and modern portfolio website showcasing projects and skills.

## 🎨 Technologies

- **CSS** (45.2%) - Styling
- **SCSS** (32%) - Preprocessor styling
- **JavaScript** (17.8%) - Interactivity
- **HTML** (5%) - Markup

## 📋 Features

- Responsive design
- Modern aesthetic
- Project showcase
- Interactive elements

## 🚀 Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Customize content with your information

## 📁 Project Structure

```
.
├── index.html          # Main HTML file
├── styles/             # CSS and SCSS files
├── js/                 # JavaScript files
└── README.md          # This file
```

## 💡 Usage

Edit the HTML, CSS/SCSS, and JavaScript files to personalize your portfolio with your projects, skills, and information.
